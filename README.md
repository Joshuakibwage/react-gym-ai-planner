## 🏋️ About The Project

Tired of generic workout routines? **Gym AI Planner** leverages the power of AI to generate personalized fitness plans tailored to your goals, experience level, and available equipment. Built with a modern, local-first tech stack, it's fast, responsive, and works even when you're offline.

Say goodbye to guesswork and hello to smarter training.

### ✨ Key Features

*   🤖 **AI-Powered Planning:** Get dynamic workout plans generated by AI to keep your routine fresh and effective.
*   ⚡ **Real-time & Local-First:** Your data is stored locally and syncs seamlessly across devices. It works perfectly offline!
*   🎯 **Goal-Oriented:** Whether you want to build muscle, lose weight, or improve endurance, get a plan that fits.
*   📈 **Track Your Progress:** (Assumed feature) Log your sets, reps, and weights to monitor your gains over time.
*   ⚛️ **Modern & Fast:** Built with React, Vite, and TypeScript for a blazing-fast user and developer experience.

This project is built with a cutting-edge, local-first stack:

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js (version 18 or higher) and a package manager (npm, yarn, or pnpm) installed.

*   **Node.js:** [https://nodejs.org/](https://nodejs.org/)

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Joshuakibwage/react-gym-ai-planner.git
    cd react-gym-ai-planner
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Set up environment variables:**

    Create a `.env.local` file in the root of the project and add your API keys.

    ```sh
    cp .env.example .env.local
    ```

    Your `.env.local` should look like this:

    ```env
    # Get your API key from https://openrouter.com/
    OPEN_ROUTER_KEY="your-openai-api-key"

    # Get your App ID from https://neon.com/
    VITE_NEON_AUTH_URL="your-instantdb-app-id"
    ```

### Running the Application

Start the development server:

```bash
npm run dev
```

Open http://localhost:5173 (or the port shown in your terminal) to view it in the browser.

### Available Scripts

*   `npm run dev`: Starts the development server.
*   `npm run build`: Builds the app for production.
*   `npm run lint`: Lints the source code.
*   `npm run preview`: Serves the production build locally.
